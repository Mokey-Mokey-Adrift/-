Викторина с загрузкой вопросов из файла
Запуск:
1. Скачайте все три файла (index.html, style.css, script.js).
2. Поместите их в одну папку.
3. Откройте index.html в любом современном браузере.

Формат JSON для загрузки
```json
[
  {
    "question": "Столица Франции?",
    "options": ["Берлин", "Мадрид", "Париж", "Рим"],
    "correctAnswer": 2
  }
]
